Bu klasördeki mevcut C++ pybind11 kodumu pip ile kurulabilir Python paketi haline getir.

Hedef:

C++ kodu Python’dan import ida_star_cpp ile import edilebilsin.
Kullanıcı fonksiyonu şu şekilde çağırabilsin:
ida_star_cpp.ida_star_cpp(lanes, log_fn=None, stop_get_best_fn=None, stop_fn=None, use_dsg_tiebreak=False, num_threads=8)
Paket lokal olarak pip install -e . ile kurulabilsin.
Sonra python -m build ile wheel/sdist üretilebilsin.

Yapılacaklar:

Proje yapısını şu hale getir:
src/ida_star_cpp.cpp
CMakeLists.txt
pyproject.toml
README.md
tests/test_import.py
.gitignore
Mevcut .cpp dosyamı src/ida_star_cpp.cpp içine taşı veya kopyala. Kodun içeriğini bozma. Özellikle PYBIND11_MODULE(ida_star_cpp, m) modül adı korunmalı.
CMakeLists.txt dosyasını scikit-build-core ve pybind11 ile uyumlu olacak şekilde yaz:
CMake minimum 3.15 olsun.
C++ standardı C++17 olsun.
pybind11_add_module(ida_star_cpp src/ida_star_cpp.cpp) kullanılsın.
Modül install edilsin.
pyproject.toml oluştur:
build backend: scikit_build_core.build
build requirements: scikit-build-core, pybind11
package name: ida-star-cpp-deniz
version: 0.1.0
requires-python: >=3.9
açıklama: Parallel IDA* solver written in C++ with pybind11
author: Deniz Efe Yıldız
README.md içine kısa kurulum ve kullanım örneği yaz:
pip install -e .
import ida_star_cpp
örnek lanes listesiyle çağırma.
tests/test_import.py içine sadece import testi ve küçük örnek çağrı testi ekle.
.gitignore içine şunları ekle:
build/
dist/
*.egg-info/
__pycache__/
.pytest_cache/
.venv/
.vscode/
*.pyd
*.so
*.dll
Sonra şu komutları sırayla çalıştır ve hata varsa düzelt:
python -m pip install --upgrade pip
python -m pip install scikit-build-core pybind11 cmake ninja build pytest
pip install -e .
python -c "import ida_star_cpp; print(ida_star_cpp)"
pytest
python -m build
Eğer Windows’ta C++ compiler hatası çıkarsa bunu açıkça belirt:
Visual Studio Build Tools / Desktop development with C++ gerekli olabilir.
